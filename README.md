# skin-disease-tg-bot
Проект на питоне для Больших Вызовов. Данный телеграмм-бот позволяет определять болезни кожи по фотографии.

Вы можете развернуть бота локально. Для этого вам необходимо:
1. Скачать test.py и new_best.pt.
2. Установить необходимые библиотеки (python-telegram-bot, ultralytics, cv2, os)
3. Создать телеграмм-бота через BotFather.
4. Полученный токент бота вставить в переменную TOKEN файла test.py.
5. Запустить test.py. Если всё успешно, то телеграмм бот начнёт работать.
